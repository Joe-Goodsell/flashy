# Flashy

Flashy is a simple, text-based flashcard app for studying and memorisation, written in Rust using Ratatui, PostgresQL and SQLX.

## Instructions
1. Clone the repo
2. `cd /path/to/flashy`
3. `./scripts/init_db.sh`
4. `cargo run`
